Projects

Undergraduate Research —
Project Sidewalk
I helped develop several new webpages for Project Sidewalk, a crowd-sourcing tool to map out and score the accessibility of different cities. I primarily used HTML, CSS, JQuery, and Scala to create a labeling guide for users.
Android Application
I created an application in Android Studio called ScoreTracker, which keeps tracks of players' scores in games, particularly card games. In addition to tracking scores, there is a small analysis of the scores at the end of each game.
Kinspire Portal
As a nonprofit organization that provides educational resources for underprivileged children in developing areas of India, I worked with a small group of people to create an interactive desktop application from start to finish. I helped with content, design (with wireframes), and development of the application.
Campus Maps
I developed a mapping service for the UW Seattle campus that finds the shortest path between two buildings. I used Java to create the directed multi-graph that stores all the paths by distance and I used Swing to create the UI.
Flight Service
I built a flight booking service that allows users to search, book, and cancel flights. I used Microsoft SQL Server through Azure SQL Database to design a database of my customers and the flights they book as well as write a Java application that connects to the database.
